# RCC Git 勉強会 2019

[![Build Status](https://travis-ci.org/0918nobita/git-workshop.svg?branch=master)](https://travis-ci.org/0918nobita/git-workshop)

GitBook を用いて作成

## 環境構築

事前に yarn の導入が必要

```bash
yarn install
```

## localhost で serve する

```bash
yarn serve
```

## HTML 形式でビルド

```bash
yarn build
```

## PDF で出力

```bash
yarn pdf
```
